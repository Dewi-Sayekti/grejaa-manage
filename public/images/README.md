Place your favicon image here as `favicon.png`.

Steps:
1. Save the image you provided (or any desired favicon) to this folder as `favicon.png`.
2. File path should be: `public/images/favicon.png`.
3. If you prefer another name, update the href in `resources/views/layouts/guest.blade.php` accordingly.
4. After replacing the image, run:

php artisan view:clear

This ensures the updated favicon is loaded.
